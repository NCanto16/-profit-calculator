Português

Este projeto foi desenvolvido com o objetivo de automatizar o cálculo de preços e lucros no comércio de roupas importadas. Antes da criação da ferramenta, todos os cálculos eram realizados manualmente, o que tornava o processo demorado, repetitivo e suscetível a erros, especialmente devido à variedade de produtos, como camisetas, bonés, tênis e calças. A aplicação resolve esse problema ao permitir que o usuário insira os valores necessários em campos específicos e, com apenas um clique, obtenha todos os resultados de forma automática.

A calculadora realiza o cálculo do frete com base no peso do produto, a soma do valor total em yuans, a conversão de yuan para real, o cálculo das taxas de importação, o desconto das taxas da plataforma de venda e, por fim, determina o preço final de venda e o lucro estimado por produto. Dessa forma, o processo de precificação se torna mais rápido, preciso e eficiente.

O projeto foi estruturado de forma modular, sendo composto por diferentes arquivos com responsabilidades específicas. O arquivo index.html contém a estrutura principal da interface, enquanto o style.css é responsável pela estilização base. O arquivo calculate.js concentra toda a lógica de cálculo e conversões, enquanto o render.js foi criado para atualizar dinamicamente os elementos do HTML com os resultados obtidos. Já o render.css é utilizado para estilizar as alterações visuais após a execução dos cálculos.

As tecnologias utilizadas no desenvolvimento incluem HTML, CSS, JavaScript e Bootstrap para a construção da interface, além do uso de Git para controle de versão. O principal objetivo do projeto é otimizar o processo de definição de preços de produtos importados, tornando-o mais ágil, confiável e escalável.

English

This project was developed with the purpose of automating price and profit calculations in the imported clothing business. Before the creation of this tool, all calculations were performed manually, making the process time-consuming, repetitive, and prone to errors, especially due to the variety of products such as t-shirts, caps, sneakers, and pants. The application addresses this issue by allowing users to input the required values into specific fields and, with a single click, obtain all results automatically.

The calculator performs shipping cost calculations based on product weight, computes the total price in Chinese yuan, converts the value from yuan to Brazilian reais, calculates import taxes, applies platform fees, and determines both the final selling price and the estimated profit per product. This significantly improves the pricing process, making it faster, more accurate, and more efficient.

The project is structured in a modular way, with different files handling specific responsibilities. The index.html file contains the main structure of the interface, while style.css is responsible for the base styling. The calculate.js file handles all calculation logic and conversions, and render.js was created to dynamically update the HTML with the computed results. Additionally, render.css is used to style the visual changes after calculations are performed.

The technologies used in this project include HTML, CSS, JavaScript, and Bootstrap for interface development, along with Git for version control. The main goal of this project is to optimize the pricing process for imported products, making it more efficient, reliable, and scalable.
